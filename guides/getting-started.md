### Getting started

1. Launch the devtool with node:
 ```
yarn global add notifme-devtool
notifme-devtool -u [app].notif.me --first
 ```
 (Or with docker: `docker run -p 1080:1080 --rm -it notifme/devtool -u [app].notif.me --first`)
